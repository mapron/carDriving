# carDriving
GUI-приложение симуляции движения автомобиля по дороге.

Зависимости:
- Qt4 >=4.8.0
- Qmake


Проверенные конфигурации сборки:
- Windows-msvc2010-x86, Qt4
- linux-gcc4.9-x86_64, Qt4

Сборка:

- Для сборки требуется система сборки QMake.
- Для подробный инструкций по сборке см. 
- http://doc.qt.io/qt-5/qmake-tutorial.html
